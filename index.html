FACULTY RESOURCES FOR COMMUNITY ENGAGED LEARNING

This is a usability test of the website for faculty resources to find the average time take by users to accomplish basic tasks and determine efficiency of website

TEST 1 :- FIND THE PURPOSE OF THE WEBSITE

TIME TAKEN: 20 SECONDS

The user managed to find the website’s purpose relatively quick within a few seconds with accuracy. The eye movements are pretty accurate and flow works right.

TEST 2 :- FIND THE CONTACT INFO FOR THE WEBSITE

TIME TAKEN: 2 MINUTES

The user struggled a bit to finish this task. User navigated through the website for quite some time due to confusion and eyeballs were all over the place. A better way would be putting contact information in the footer

TEST 3 :- FIND EXAMPLE SYLLABI FOR ENV SCIENCE

TIME TAKEN: 4 MINUTES

The user took a lot of time with this one. There was a lot of confusion and lot of navigation. Had to use the search bar at the end to find. A suggestion would be having shorter headings or sub-sections in the menu bar.

TEST 4 :- FIND THE COMMUNITY ENGAGED FACULTY

TIME TAKEN: 10 Seconds

The user managed to find the list of faculty members very quickly. There was not much trouble.

TEST 5 :- FIND RESOURCES ON ENGAGED SCHOLARSHIPS

TIME TAKEN: 10 Seconds

The user managed to find the engaged scholarships rapidly with accuracy.

What is the SUS?
The SUS provides a “quick and dirty”, reliable tool for measuring the usability.It consists of a 10 item questionnaire with five response options for respondents.
On a scale of 1 to 5... (where 1 is strongly disagree and 5 is strongly agree)
